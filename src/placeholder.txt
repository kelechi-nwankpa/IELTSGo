TECHNOLOGY STACK
----------------

Frontend:
- Next.js 14+ (App Router)
- React 18+
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod (validation)

Backend:
- Next.js API Routes
- Prisma (ORM)
- PostgreSQL
- Redis (caching)

AI Integration:
- Anthropic Claude API
- OpenAI Whisper API (speech-to-text)

Authentication:
- NextAuth.js or custom JWT

Payments:
- Stripe
